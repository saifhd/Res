Installing method
	1-> create database and name it as 'res'
	2-> move res folder to xampp/htdocs folder
	3-> run the cmd and set path to 'res' folder
	4-> run the command on cmd --> php artisan migrate
	5-> put some datas to events table
	6-> in navigation bar events tab only working
